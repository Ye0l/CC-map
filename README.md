# CC-Map Discord Bot 도움말

파이널 판타지 14 크리스탈 컨플릭트 맵 정보 및 부가 기능을 제공하는 디스코드 봇입니다.

[🤖 봇 초대하기](https://discord.com/oauth2/authorize?client_id=1462062834262343753&permissions=18172928&integration_type=0&scope=bot+applications.commands)

## 📖 명령어 목록

모든 명령어는 슬래시 커맨드(`/`)로 실행됩니다. 한글 별칭으로도 사용할 수 있습니다.

### 1. 맵 정보 확인

- **/now (지금)**
  - 현재 진행 중인 크리스탈 컨플릭트 맵 정보를 보여줍니다.
  - 종료 시간과 다음 맵 정보를 확인할 수 있습니다.

- **/rotation (로테이션)**
  - 향후 맵 로테이션 일정을 확인합니다.
  - **옵션**:
    - `count (개수)`: 표시할 향후 일정의 개수 (최대 10개, 기본 5개)

- **/when (언제)**
  - 특정 맵이 언제 나오는지 검색합니다.
  - **옵션**:
    - `map_name (맵이름)`: 검색할 맵 이름 (필수, 자동완성 지원)
    - `count (개수)`: 표시할 해당 맵의 시간표 개수 (기본 1개)

### 2. 운세 및 추천

- **/horoscope (운세)**
  - 오늘의 별자리 운세를 확인합니다. (FF14 테마)
  - 운세와 함께 추천 맵 및 직업을 알려줍니다.
  - **옵션**:
    - `sign (별자리)`: 본인의 별자리 (필수)

- **/recommend (직업추천)**
  - 무작위로 PvP 직업을 추천해줍니다.
  - **옵션**:
    - `count (개수)`: 입력 시 해당 개수만큼 직업을 간략하게 목록으로 보여줍니다. (연속 가챠 모드)

### 3. 미니게임 및 유틸리티

- **/dice (주사위)**
  - 주사위를 굴립니다.
  - **옵션**:
    - `range (범위)`: 주사위 범위 (예: 1-100). 기본값은 1-999입니다.

- **/duel (결투)**
  - 다른 사용자와 주사위 결투를 합니다.
  - **옵션**:
    - `target (상대방)`: 결투를 신청할 상대방 (필수)

- **/stats (전적)**
  - 결투 전적을 확인합니다.
  - **옵션**:
    - `target (대상)`: 전적을 확인할 대상 (생략 시 본인)

- **/team (팀)**
  - 인원이나 이름을 두 팀(A팀/B팀)으로 무작위로 나눕니다.
  - **옵션** (둘 중 하나 필수):
    - `count (인원)`: 인원 수 (1부터 해당 숫자까지 배분)
    - `names (이름)`: 공백으로 구분된 이름 목록

- **/tip (팁)**
  - 저장된 유용한 팁을 키워드로 검색합니다.
  - tip은 WebUI 관리자 페이지에서 추가할 수 있습니다.
  - **옵션**:
    - `keyword (키워드)`: 검색할 팁 키워드 (필수)


### 4. 질문 및 챗봇 (Beta)

- **/ask (질문)**
  - 크리스탈 컨플릭트 기술이나 전략 등에 대해 챗봇에게 질문합니다.
  - **옵션**:
    - `content (내용)`: 질문할 내용 (필수)


### 5. 맵 이미지 설정 (관리자용)

`/now` 및 `/when` 명령어에서 맵 이미지를 함께 보여주려면 `maps` 폴더에 이미지를 추가해야 합니다.

1. 봇이 실행되는 위치에 `maps` 폴더를 생성합니다.
2. 각 맵에 해당하는 이미지를 아래 이름으로 저장합니다. (확장은 `.png` 권장)
   - 팔라이스트라: `palaistra.png`
   - 화산심장: `volcano.png`
   - 동방 꼭두각시 어전: `clockwork.png`
   - 바닷가 전투장: `seaside.png`
   - 절정의 구름: `cloud.png`
   - 붉은 사막: `desert.png`
